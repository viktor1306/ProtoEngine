# Модуль Core (`src/core`)

Цей модуль містить незалежні від платформи утиліти та системи, що складають основу рушія.

## Ключові Компоненти

### `Window` (`Window.hpp/cpp`)
- Обгортка над WIN32 API для створення та керування вікном програми.
- Обробляє системні повідомлення (зміна розміру, закриття).
- **Не обробляє введення**: Обробка введення винесена в `InputManager`.

### `InputManager` (`InputManager.hpp/cpp`)
- Сінґлтон клас для опитування стану введення.
- Підтримує клавіатуру (WASD) та мишу (Delta переміщення).
- Потокобезпечні оновлення (викликається один раз на кадр).

### `Math` (`Math.hpp`)
- Легковагова математична бібліотека (header-only).
- Надає `Vec2`, `Vec3`, `Vec4`, `Mat4`.
- Оптимізована для 3D графіки (матриці проекції, lookAt, трансформації).

### `ShaderHotReloader` (`ShaderHotReloader.hpp/cpp`)
- **Функція**: Дозволяє редагувати файли шейдерів (`.vert`, `.frag`) під час роботи рушія.
- **Механізм**:
    - Слідкує за часом зміни файлів у окремому потоці (перевірка кожні 500мс).
    - Компілює GLSL у SPIR-V, використовуючи `glslc` (через `_popen`).
    - Сигналізує `Renderer` про необхідність безпечного перестворення пайплайнів.
- **Безпека**: Перехоплює помилки компіляції та запобігає падінню програми, залишаючи старі пайплайни активними у разі невдачі.
