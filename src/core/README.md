# Модуль Core (`src/core`)

Цей модуль містить незалежні від платформи утиліти та системи, що складають основу рушія.

## Ключові Компоненти

### `Window` (`Window.hpp/cpp`)
- Обгортка над WIN32 API для створення та керування вікном програми.
- Обробляє системні повідомлення (зміна розміру, закриття).
- Надає `getHWND()` та `getHINSTANCE()` для Vulkan surface creation.
- **Не обробляє введення**: Обробка введення винесена в `InputManager`.

### `InputManager` (`InputManager.hpp/cpp`)
- Сінґлтон клас для опитування стану введення.
- Підтримує клавіатуру (`WASD`, `Esc`) та мишу (delta переміщення, ПКМ).
- `update()` — викликається один раз на початку кожного кадру.

### `Math` (`Math.hpp`)
- Легковагова математична бібліотека (header-only, ~157 рядків).
- Надає: `Vec2`, `Vec3`, `Vec4`, `Mat4`.
- Операції: `Mat4::translate`, `Mat4::rotate`, `Mat4::scale`, `Mat4::lookAt`, `Mat4::perspective`, `Mat4::ortho`.
- Оптимізована для 3D графіки (column-major матриці, Vulkan NDC).

### `ShaderHotReloader` (`ShaderHotReloader.hpp/cpp`)
- Дозволяє редагувати файли шейдерів (`.vert`, `.frag`) під час роботи рушія.
- **Механізм**:
  - Слідкує за часом зміни файлів у окремому потоці (перевірка кожні 500 мс).
  - Компілює GLSL у SPIR-V через `glslc` (або `glslangValidator`).
  - Сигналізує `Renderer` про необхідність безпечного перестворення пайплайнів.
- **Безпека**: Перехоплює помилки компіляції та залишає старі пайплайни активними у разі невдачі.
- API: `watch(path)`, `start()`, `shouldReload()`, `ackReload()`.
