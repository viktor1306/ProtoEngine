# Модуль UI (`src/ui`)

Системи користувацького інтерфейсу та рендерингу тексту.

## Ключові Компоненти

### `TextRenderer`
- Рендерить 2D текст поверх сцени.
- Використовує **Signed Distance Fields (SDF)** для чіткого тексту при будь-якому масштабуванні.
- **Batching**: Використовує динамічний staging буфер для завантаження вершин текстових квадів.
- Інтегрований з `BindlessSystem` для семплювання атласу шрифту.

### `FontSDF`
- Генерує атласи SDF текстур з TrueType шрифтів (`.ttf`).
- Використовує `stb_truetype` для растеризації.
- Кешує метрики гліфів для правильного розташування тексту.
